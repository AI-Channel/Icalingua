# Icalingua

Icalingua 这个名字是 [@91khr](https://github.com/91khr) 起的，是拉丁语中「光」和「语言」的组合。

本项目希望为 Linux 打造一个会话前端框架，通过实现 Adapter 后端接口来适配各种聊天平台。目前已经拥有基于 [oicq](https://github.com/takayama-lily/oicq) 的后端
